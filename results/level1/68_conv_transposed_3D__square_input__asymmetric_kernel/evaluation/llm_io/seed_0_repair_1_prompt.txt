      - groups = 1
      - contiguous NCDHW layout
    Falls back to PyTorch for other configurations.

        stride != (1, 1, 1)
        or padding != (0, 0, 0)
        or output_padding != (0, 0, 0)
        or groups != 1

            x, weight, bias, stride=stride, padding=padding,
            output_padding=output_padding, groups=groups
